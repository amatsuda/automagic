= Automagic

This gem makes your Ruby automatically inject the "magic comment" into your .rb files.

== Supported

* Ruby 1.9
* Ruby 2.0


==  Not Supported

* require_relative


== Contributing to Automagic

* Fork, fix, then send me a pull request.
